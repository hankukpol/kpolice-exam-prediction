### 1-2. 경쟁사 공통 운영 패턴

```
[시험 3~4주 전]  사전예약 이벤트 오픈 + SNS/카페 소문내기
[시험 당일]      필기 종료 직후 → 풀서비스 오픈 + 1차 합격컷 + 라이브 총평
[시험 후 1~5일]  2차·3차·4차 합격컷 순차 공개 (참여자 증가 → 정확도 향상)
[합격 발표 전후]  정확도 검증 홍보 → 면접/체력 대비 강좌 전환
```

### 1-3. 경쟁사 수익 모델

모든 경쟁사의 합격예측은 **무료**. 수익 구조:
1. **리드 수집**: 회원가입을 통한 수험생 개인정보(이름, 연락처, 지역) 확보
2. **브랜드 노출**: 해설강의·총평으로 강사/학원 인지도 강화
3. **수강 전환**: 면접·체력 대비 유료 강좌로 자연스러운 전환
4. **재방문 유도**: 합격컷 업데이트, SMS 알림으로 반복 방문
5. **패스 환급 연계**: 기존 수강생에게 참여를 환급 조건으로 설정

### 1-4. 수험생 커뮤니티 반응 (니즈 분석)

**핵심 니즈**:
- "시험 끝나자마자 바로 점수 확인하고 싶다" → **즉시성**
- "내가 몇 배수인지 빨리 알고 싶다" → **합격 가능성 확인**
- "대구 공채 합격컷이 얼마인지" → **지역별 정밀 분석**
- "나보다 위에 몇 명이 있는지" → **경쟁자 비교**

**불만 사항**:
- "대형 사이트는 서울은 정확한데 **지방은 참여자가 적어서 부정확**하다"
- "1차 합격컷은 믿을 수 없다, 3~4차 되어야 정확해진다"
- "채점만 하면 되는데 회원가입해야 한다"
- "강좌 광고가 너무 많다"

**2026년 특이사항**:
- 남녀 **통합선발 첫 해** → 기존 데이터 기반 예측이 무의미, 불확실성 극대화
- 수험생 69%가 합격컷 상승 예상 → 합격예측 수요 사상 최대

## 2. 우리 사이트 SWOT 분석

### 강점 (Strengths)

| # | 항목 | 설명 |
|---|------|------|
| S1 | **지역 밀착 데이터** | 대구·경북 지역 학원생 데이터로 해당 지역에서는 대형 사이트보다 정확할 수 있음 |
| S2 | **광고 없는 깔끔한 UI** | 강좌 광고·패스 상품 배너 없이 순수 합격예측에 집중 |
| S3 | **완성도 높은 관리도구** | 배너·이벤트·공지·합격컷 발표·재채점 등 10개 이상 관리 모듈 |
| S4 | **실시간 커뮤니티** | 댓글 시스템 (30초 폴링) + 닉네임 마스킹 |
| S5 | **정확한 채점 엔진** | 공채/경행경채 분리, 가산점 상호배제, 자동 재채점 |
| S6 | **단계적 합격컷 발표** | 1~4차 합격컷 발표 + 스냅샷 히스토리 기능 구현 완료 |
| S7 | **오프라인 학원 연계** | 대구경찰학원 수강생 직접 안내 가능 |

### 약점 (Weaknesses)

| # | 항목 | 설명 |
|---|------|------|
| W1 | **참여자 수 부족** | 대형 사이트 대비 초기 데이터 확보 어려움 |
| W2 | **브랜드 인지도** | 전국 수험생 대상 인지도 낮음 |
| W3 | **해설강의 없음** | 시험 직후 해설·총평 콘텐츠 미제공 |
| W4 | **마케팅 예산 제한** | 네이버페이·경품 등 금전 인센티브 제공 어려움 |
| W5 | **푸시 알림 미지원** | SMS/앱 푸시 알림 없음 (웹 전용) |

### 기회 (Opportunities)

| # | 항목 | 설명 |
|---|------|------|
| O1 | **통합선발 첫 해** | 기존 데이터 무의미 → 모든 업체가 동일 출발선, 정확도 격차 줄일 기회 |
| O2 | **지방 수험생 불만** | 대형 사이트의 지방 정확도 불만 → 대구·경북 특화 포지셔닝 |
| O3 | **간편 접근 니즈** | 회원가입 강제에 대한 수험생 불만 → 간편 가입으로 차별화 |
| O4 | **학원 수강생 기반** | 기존 학원 수강생 확보 → 초기 데이터 시드 역할 |

| # | 항목 | 설명 |
|---|------|------|
| T1 | **빅데이터 선순환** | 대형 사이트의 "많은 참여자 → 높은 정확도 → 더 많은 참여자" 구조 |
| T2 | **경품 경쟁** | 해커스 네이버페이 1만원 등 금전 인센티브 경쟁 |
| T3 | **수험생 다중 이용** | 대부분의 수험생이 2~3개 사이트에 동시 참여 |

## 3. 포지셔닝 전략

### 핵심 가치 제안 (USP)

> **"대구·경북 지역 1위 경찰학원이 만든, 대구·경북 특화 합격예측"**

대형 사이트와 정면 경쟁하지 않고, **지역 특화**로 차별화:

| 대형 사이트 | 우리 사이트 |
|------------|-----------|
| 전국 데이터 (서울 편중) | **대구·경북 밀착 데이터** |
| 강좌 광고가 많은 복잡한 UI | **광고 없는 깔끔한 분석 도구** |
| 회원가입 필수 + 개인정보 수집 | **간편 가입 (연락처 + 비밀번호)** |
| 전국 평균 기반 예측 | **지역별 정밀 예측** |

### 보조 메시지

- "대구·경북 수험생을 위한 합격예측"
- "광고 없이 순수하게, 내 합격 가능성만 확인"
- "대구경찰학원 11년 노하우가 담긴 합격 분석"

#### 확정답안 변경 시 운영 절차

1. 공식 확정답안 확인 (경찰청 발표)
2. /admin/answers에서 변경된 문항 수정 + "확정" 체크
3. 변경 사유 입력 (예: "3번 문항 정답 변경 2→3")
4. **자동 재채점** 실행 → 전체 제출자 점수 재계산
5. 사용자에게 **재채점 알림 모달** 자동 표시
6. 합격컷 재발표 (/admin/pass-cut)
7. 공지 등록: "확정답안 반영 완료, 점수가 변경된 분은 다시 확인해 주세요"

### 5-1. 사용자 확보 전략 (현실적 실행 가능한 방안)

#### 1차 타겟: 학원 수강생 (확보 용이, 핵심 시드 데이터)
